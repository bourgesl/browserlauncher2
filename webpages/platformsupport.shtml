Platform Support for BrowserLauncher2

OS Name Browsers Status Browser Targetting Notes
Linux (2.4.x or higher) FireFox, Mozilla, Netscape, Konqueror, Opera Supported Supported If more than one of the supported browsers is available, the first one in alpha order will be invoked as the default.
Mac OS X (10.2 or higher) Supported Not supported, calls default
Windows 2000 (5.0) Mozilla, Firefox, IE Supported Partially supported. Users can pass the name of a browser to target but the list of available browsers may not be accurate.
